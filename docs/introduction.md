# Introduction

This document is intended to provide guidance for testing Accessibility issues for products that will not be using the Level Access Platform to conduct both automated testing using a browser plug-in as well as a manual evaluation.

::: warning
Achieving WCAG 2.1 Level AA (ADA Title II) compliance requires completing all three steps: automated testing, manual testing, and full remediation of every issue found.

- Automated testing
- Manual testing
- Full remediation of every issue found
:::
